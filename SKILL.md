# aaa-pm

Container skill for the AAA PM pack. All skills use the **`aaa-pm-`** prefix.

1. Run **`/aaa-pm-about`** — layout, workflow, skill list.
2. Run **`/aaa-pm-setup`** — scaffold **`docs/pm/`** in the product repo (any project).

Independent of app-specific skills (`@ceo-*`, `docs/qa/`, etc.).
