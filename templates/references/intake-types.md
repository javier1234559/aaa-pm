# Intake classification

| Type | When | Brief impact | Jira on push |
|------|------|--------------|--------------|
| BUG | Wrong vs acceptance / regression | Usually small | Bug |
| FEAT | New capability | Medium–large | Story/Task |
| CHANGE | Requirement shift | Large → re-plan | Story + update `project-brief` §Features |
| QUESTION | Unclear, no build yet | §Open questions only | None |
| CHORE | Tech debt, CI, docs | Small | Task |

**Large impact** (any): changes §Goal, moves Must→Won't, new epic theme, >3 stories affected → recommend **`/aaa-pm-plan`** before push.
